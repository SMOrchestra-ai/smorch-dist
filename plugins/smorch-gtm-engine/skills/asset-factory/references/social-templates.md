## Social Post Templates

### Architecture

3 posts per campaign. Each targets a different ICP segment with a different psychological trigger as primary driver.

| Post | ICP Target | Primary Trigger | Secondary Trigger | CTA Type |
|------|-----------|----------------|-------------------|----------|
| 1 | Pipeline problems (has product, broken GTM) | Identity Threat | Costly Signal | Assessment link |
| 2 | No audience (has expertise, no distribution) | Costly Signal | Identity Threat | Assessment link |
| 3 | Founder bottleneck (founder IS the GTM) | Temporal Window | Identity Threat | Assessment link |

### Format Rules

- 150-210 words per post
- Opens with identity threat hook (competitor advantage angle)
- Includes anonymized case example with specific numbers
- Embeds costly signal naturally (guarantee, methodology proof)
- Temporal window in CTA (market timing, not fake countdown)
- Copy-paste ready (no cleanup needed)
- No hashtags in body. Add 2-3 at bottom if platform requires

**Primary trigger**: Identity Threat

```
[Anonymized case]: [Company descriptor, geography] was stuck at [revenue/metric]. Running [old method number] [old methods]. [Timeframe] deal cycles.

The pattern from [number] companies studied:

The ones stuck run [low number] [motions/strategies].
The ones scaling run [higher number], [key differentiator].

This isn't about [conventional solution].
It's about [engineered alternative].

[Guarantee/proof statement as costly signal].

[Temporal window]: [Market timing context]. Every [time unit] you spend on [old method], [consequence with competitor framing].

[Low-friction CTA with specific action and time commitment]
```
