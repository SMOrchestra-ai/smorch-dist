## Comments Strategy

**Why put the link in a comment, not the post?**
1. LinkedIn algorithm doesn't penalize comments with links as heavily as post body links
2. Comment with link stays visible at top when sorted by "Top Comments"
3. Post stays clean and focused on value, not CTA
4. Higher click-through rate on comment link vs post link

**Comment timing:**
- Post the main content
- Within 60 seconds, post the comment with the registration link
- Check back after 1 hour—your comment should be pinned at top
