## FORMATTING RULES

### Line Structure
1. Max 55 characters per line (phone-first)
2. Line break after every THOUGHT, not every sentence. A thought can be 3 words.
3. Fragments encouraged. "No CTO. No funding. No excuses." Three beats.
4. Hook is exactly 2 lines before "see more" fold. Statement, never question.

### Post Architecture
```
[Hook Line 1: Bold claim, result, or accusation]
[Hook Line 2: Contrasting twist or specificity]

[1-2 line context: the problem or setup]

[Proof: specific number, timeframe, tool name]

[Breakdown: numbered steps, old/new, or named framework]

[Disagreement trigger: one claim that splits the audience]

[Thesis reinforcement: one declarative sentence]

[CTA: soft. Value speaks. CTA whispers.]
```

### Sentence Construction
1. Subject-verb-object. Active voice only.
2. Rarely more than 8 words per sentence.
3. Repetition as emphasis: three-beat patterns.
4. Numbers always digits. Always specific. Always with timeframes.
5. Tool names always named. Never "an enrichment tool." Always "Clay."
6. Contrarian claim BEFORE the framework. Break belief, then teach replacement.
7. Every sentence teaches, proves, or provokes. Cut everything else.

### Emoji Rules
- Maximum 2 per post. Structure emojis only (numbered lists, arrows).
- Never: clapping, fire, rocket, 100, prayer hands.

## ENGAGEMENT MECHANICS (Mandatory in every post)

### 1. Disagreement Trigger
Every post MUST contain one claim reasonable people could argue against. Disagreement drives comments.

Place after you've built credibility with proof. Never open with unsubstantiated controversy.

### 2. Pinned Comment (Comment Magnet)
Write alongside every post. The pinned comment is the MOST valuable piece:
- A bonus tool not in the main post
- A link to a resource or template
- An expanded version of one step
- A "next week I'll cover..." teaser

Format: 2-4 sentences. Value-dense. Zero self-promotion.

### 3. Self-Identification Prompt
End with something that forces reader to see themselves:
- "If you're still [doing old thing], this was for you."
- "Which camp: [A] or [B]?"
- "Comment [word] if you want the template."

ONE per post. Rotate. Never repeat same prompt in consecutive posts.

### CTA Rotation (never same twice in a row)
1. Self-ID prompt (comment engagement)
2. P.S. question format
3. "Save this. You'll need it." (algorithm signal)
4. "Link in comments" (for resources)
5. "Full deep dive: entrepreneursoasis.me" (Track A only, max 1x/week)
6. "Curious? Drop a comment." (engagement play)

## HOOK PSYCHOLOGY (Both skills reference this)

### 8 Psychological Triggers

1. **CONTRADICTION** - Sounds wrong on purpose. Forces "wait, what?"
2. **ACCUSATION** - Calls reader out directly. Creates defensiveness that demands click.
3. **STOLEN THOUGHT** - Says what they think but won't say publicly. "Finally someone said it."
4. **SPECIFICITY** - Number so precise it can't be fabricated. Instant credibility.
5. **STORY ENTRY** - Drops you into a moment. "What happened next?"
6. **TIMELINESS** - First to report. Urgency and FOMO.
7. **COMPARISON** - Old vs New or Tool A vs Tool B. Forces opinion.
8. **SYSTEM REVEAL** - Exposes the machinery. "I need to save this."

Pick the trigger FIRST, then write the hook.
