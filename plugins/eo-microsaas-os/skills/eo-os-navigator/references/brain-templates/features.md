## 4. User Flow | مسار المستخدم

**Describe the happy path in 5 steps or less:**
وصف المسار الأساسي بـ 5 خطوات أو أقل:

```
Step 1: User signs up / يسجل
Step 2: ...
Step 3: ...
Step 4: ...
Step 5: User gets value / يحصل القيمة
```

**How fast does the user get to the "aha moment"?**
كم يأخذ المستخدم وقت لين يوصل لحظة "آها!"؟

- [ ] Under 2 minutes أقل من دقيقتين
- [ ] Under 10 minutes أقل من 10 دقايق
- [ ] Under 1 hour أقل من ساعة
- [ ] Requires setup/onboarding يحتاج إعداد

---

## 5. Technical Simplicity Check | فحص البساطة التقنية

**Can a solo founder build this MVP in a weekend with AI tools?**
يقدر مؤسس لوحده يبني هالـ MVP بعطلة نهاية أسبوع باستخدام أدوات AI؟

- [ ] Yes نعم
- [ ] Yes, but needs one integration نعم، بس يحتاج integration واحد
- [ ] No, needs multiple complex integrations لا، يحتاج integrations معقدة
- [ ] No, needs specialized infrastructure لا، يحتاج بنية تحتية متخصصة

If the answer isn't "Yes," simplify until it is.
إذا الجواب مو "نعم"، بسّط لين يصير نعم.
